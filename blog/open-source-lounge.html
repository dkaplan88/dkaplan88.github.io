Open Source Lounge Continues to Thrive

By Dan Kaplan, Sr. Software Engineer & Culture Guild Leader • April 7, 2015

The blog post discusses the evolution of Enova's Open Source Lounge, a collaborative event focused on open source projects and teamwork.

Key Learnings

Two key learnings from initial events:

  1. Attendees responded well to specific directives
  2. Participants enjoyed pairing up to solve problems

December Open Source Lounge

The December event focused on the Rubinious Project:

  • Led by Brian Shirai, a core Rubinious contributor
  • Rubinious is a Ruby implementation designed for concurrency
  • Participants worked on bug fixes and implementing Rubinious support

February Open Source Lounge

The February event centered on HeartBeat:

  • A tool used to measure workplace values and happiness
  • Attendees worked together on new features

Future Plans

Future plans include:

  • Making the Open Source Lounge a monthly event
  • Making it a public event to showcase Enova's commitment to open source

Stay tuned for more updates!
